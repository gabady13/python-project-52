## Описание

Task Manager — учебный веб-проект на Django: управление задачами с авторизацией и разграничением прав.

Функциональность:
- регистрация и аутентификация пользователей
- CRUD для статусов, меток и задач
- назначение исполнителя
- фильтрация задач
- правила удаления (например, только автор может удалить задачу)

## Требования

- Python 3.13+
- uv (рекомендуется) или совместимое окружение Python
